# Live Datamoshing

An experimental project for real-time datamoshing effects based on this [tutorial](https://www.youtube.com/watch?v=w8c33t2CgtA).

## Usage

1. Open `live_datamosh.toe`.
2. Select your camera input.
3. Use the **1** and **2** keys on your keyboard to interact with the effect.

This setup is a work in progress and will be expanded for further creative use.

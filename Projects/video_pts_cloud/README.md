# Video Point Cloud

A setup for converting video data into interactive 3D point clouds in TouchDesigner.

## Contents

- `video_pts_cloud.toe`: Main project file for point cloud generation.
- `video_point_cloud.tox`: Exported component for use in other setups.

## Usage

Connect a video source or use the provided test media to see the point cloud generation. Adjust point density and spatial parameters within the network.

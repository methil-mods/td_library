# Dithering Project

A project dedicated to exploring and developing dithering techniques in TouchDesigner.

## Contents

- `Dithering.toe`: Main project file for dithering development.
- `datas/`: Folder containing example matrix patterns and textures for dithering.

## Usage

Connect your image or video source to the dithering setup. You can swap matrices from the `datas` folder to experiment with different visual patterns.
